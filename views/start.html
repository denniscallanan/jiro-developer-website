Creating a New App

Before you create a Jiro app you must download the Jiro console on your computer, and the Jiro controller on your phone.

Next, open up the Jiro installation directory on your computer, and open the apps folder.

Create a new folder for your app. We recommend the following naming convention: authorName.appName

Inside this folder, create a new file called japp.json. This file will store info about your app:

{ "name": "The name of your app", "author": "The author of your app", "iconUrl": "http://some.url.com/for-your-icon.png", "run": "app.py" }

Change "run" to the name of the file that you would like to be executed to start your app. This may be a python file at first, but when your game is being submitted it will be an .exe file.



That's it! You're ready to create your first app! See the links under the Learning Jiro section on the left to learn how to integrate jiro with your app.
