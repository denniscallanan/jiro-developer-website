Python API

Getting Started

Once you've downloaded the API for python you should be able to import the module:

import jgsapi

You must now create a game server that your phone can connect to.

jiro = jgsapi.GameServer()

That should be enough to connect to the game from your phone.

Server options

You can pass server options into the GameServer constructor.

jiro = jgsapi.GameServer({'max-players': 8})

Controllers

Importing controllers

Jiro uses XML files to create controllers.

If you haven't created a controller yet you can follow the tutorial LINK::here

Assuming your controllers are in the "controllers" directory, you can import them by doing the following:

jiro.importController("controllers/mycontroller1.xml") jiro.importController("controllers/mycontroller2.xml") # etc.

Switching controllers

To switch controllers, use the following code:

jiro.switchController("mycontroller1")

The controller id (defined in the XML document) is passed into the function.

You can also change the controller of a specific player or a list of players.

jiro.switchController("mycontroller1", somePlayer) jiro.switchController("mycontroller2", [somePlayer, someOtherPlayer, anotherPlayer])

Controller bindings

Your controllers may contain bindings.

Get bindings allow you to retrieve xml information from your controller and set bindings allow you to modify the attributes and content of controller elements.

Link::Click here to learn how controller bindings work in XML.

Set bindings

<controller id="example"> <row bgcolor="#eee" setbinding="somerow"> <text setbinding="sometext">This is some text!</text> </row> </controller>

Pass in the controller and binding names into the following function to get the binding.

textbinding = jiro.binding("example", "sometext")

Use the following code to change the content of a binding.

textbinding.setContent("This is some new text!")

This is how to change the attributes of a binding:

rowbinding = jiro.binding("example", "somerow") rowbinding.setAttribute("bgcolor", "#ff0000") #red

You can also change binding data of a specific player.

textbinding.setContent("Hello, Player!", somePlayer) rowbinding.setAttribute("bgcolor", "#00ff00", somePlayer) #green

Get bindings

<controller id="example"> <row bgcolor="#eee" getbinding="somerow"> <text getbinding="sometext">This is some text!</text> </row> </controller>

You can get get bindings the same way you get set bindings: (that's a tongue twister!)

mybinding = jiro.binding("example", "sometext")

We can now access the content of that element.

print mybinding.getDefaultContent(); # output: This is some text!

You can also get the attributes of an element.

mybinding = jiro.binding("example", "somerow") print somerow.getDefaultAttribute("bgcolor") # output: #eee

The above examples only retrieve the default data in the XML file. If the data gets changed using set bindings we must access the data like this:

mybinding.getCurrentContent() mybinding.getCurrentAttribute("bgcolor")

To retrieve binding data from a specific player you can do this:

mybinding.getCurrentContent(player) mybinding.getCurrentAttribute("bgcolor", player)

Two-way bindings

You can get two-way bindings the same way as any other binding:

mybinding = jiro.binding("controllerID", "bindingName")

You can both get and set data with two-way bindings.

Look at get bindings and set bindings above for examples.

Event handlers

Event handlers are very important when you are creating games.

The Jiro API contains many events that you can listen to.

Button Events

There are 5 events available for buttons:

"tap" "tapStart" "tapEnd" "holdStart" "holdEnd"

Before you add an event listener to a button you must retrieve it using the controller and button ids.

<controller id="mycontroller"> <row> <button id="mybutton"><text>Tap me!</text></button> </row> </controller> button = jiro.getInteractable("mycontroller", "mybutton")

Now that that's done you should be able to add event listeners.

To add an event listener call the addEventListener function with the event type and a callback.

button.addEventListener("eventType", callback)

Example:

def tapEvent(event): print event.player.displayName + " just tapped me!" button.addEventListener("tap", tapEvent)

Accelerometer Events

Before you can add accelerometer events, you must retrieve an interactable instance of your controller.

<controller id="mycontroller" interact-accelerometer> ... </controller> mycontroller = jiro.getInteractable("mycontroller")

To add an event that checks for a change in the accelerometer, use the following code.

def accelerometerChange(event): # TODO mycontroller.addEventListener("accelerometerChange", accelerometerChange)
