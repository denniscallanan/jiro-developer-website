XML Controller

Jiro uses the XML format to create controllers for your game.

In this topic we will be learning how to create our own controllers.

Begin by creating a new xml file for your controller, and change the id to whatever you wish.

<controller id="main"> </controller>

The API allows you to switch controllers by id.

Rows & Columns

Rows and columns are used to position elements and change the size of their contents. They can be nested.

The span of each row or column defaults to a weight of 1, but you can change this using the span tag.

<controller id="main"> <row> </row> <row span="2"> <col span="3"> </col> <col span="2"> </col> </row> </controller>

On rows you can also set span="content" which sets its height to the height of its content.
This feature will be also added to columns in the future.

Text

Use the text tag inside a cell (row or column) to add text.

<controller id="main"> <row> <text size="24" weight="bold" color="#ff0000"> This is some bold text with a font size of 24. All the weight options are "thin", "light", "regular", "medium", "bold" and "black". </text> </row> </controller>

Default values:

  • size="DENNIS TELL ME!"
  • weight="regular"
  • color="#000000" (black)

Buttons

To add a button use the button tag inside of a cell and add a button id.

<row> <button id="myButton"> <text> This is a button </text> </button> </row>

The API allows you to hook up tap handlers to buttons and other elements. See here.

Images

Images are very easy to add. They stay centered inside its cell and you can use the scale attribute to change their sizes.

<controller id="main"> <row> <img src="http://some.link.com/to-an-image.png" scale="0.5"></img> </row> </controller>

You can set scalerel="cell" to make the image scale relative to its cell size instead of the default scalerel="image" which is relative to the image size.

Images will always keep their aspect ratio.

Styling

Add a background color to a cell:

<row bgcolor="#eee"> ... </row>

Add a background image to a cell:

<row bgimage="image"> ... </row>

Add padding to a cell:

<col padding="4"> ... </col>

You can set the padding for individual sides aswell (top, left, bottom, right)

<col padding="4 0 2 0"> ... </col>

Add a border to a cell:

<row bordersize="1" bordercolor="#00ff00"> ... </row>

Interactions

Interactions allow you to add event listeners to elements in your controller using the API.

To make any element interactable, simply add an id attribute.

<row id="someRowID"> ... </row> <button id="someButtonID"> ... </button>

You've reached the end of this article and should now have learned everything you need know to create a controller for your Jiro app!
